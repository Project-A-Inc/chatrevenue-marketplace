# /hello — ChatRevenue greeting skill

When the user invokes `/hello`, respond:

- With no argument: `Hello from ChatRevenue! The chatrevenue-hello plugin is installed and working.`
- With a name argument (e.g. `/hello Sasha`): `Hello, Sasha! From the ChatRevenue marketplace.`

Keep the reply to one short sentence. Do not add follow-up questions.
